# SEO NeuroAI Mobile Frontend

Frontend mobile em React Native com Expo para o SEO — Sistema de Eficiência Operacional.

## Como correr

```bash
cd frontend
npm install
npm start
```

## Ligação ao backend

Por defeito, o app espera a API em:

```text
http://127.0.0.1:8000
```

Em telemóvel físico, troca para o IP local do computador, por exemplo:

```text
http://192.168.1.10:8000
```
